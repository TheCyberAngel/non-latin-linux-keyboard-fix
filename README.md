Fixes keyboard input and key rebinding issues on Linux systems when using non-Latin keyboard layouts in Barotrauma.

This client-side LuaCs mod restores proper key detection for:
• Ukrainian
• Russian
• Belarusian
• Greek
• Bulgarian
• Serbian Cyrillic
• Macedonian Cyrillic
• Kazakh Cyrillic
• Mongolian Cyrillic
• Tajik Cyrillic
• Armenian Phonetic
• Georgian QWERTY

The mod is client-side only and works on servers without requiring server installation.

Recommended for Linux players experiencing:
• movement keys not working
• rebinding resulting in "None"
• broken WASD on Cyrillic/Greek layouts

This is a workaround for SDL/MonoGame layout-dependent keyboard mapping behavior on Linux.
